linear search - it involves iterating through each element in the data set sequentially until the target avlue is found
or the entire data set is traversed

Space complexity : - O(1) since we don't take any space other than the array/dataset (only iterating on the dataset) and since it's n (constant) hence O(1)
can't implement on data structures that are not continuous memory block to work on (sequential memory/data)
non sequential eg trees graphs etc
